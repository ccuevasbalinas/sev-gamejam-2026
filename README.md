## About the Project

**Skull Dungeon** is a game developed as part of the **Sevilla Game Jam 2026**.

Created within the limited timeframe of the jam (48h), the project focuses on delivering an engaging gameplay experience while exploring original mechanics, rapid prototyping, and collaborative game development.

This repository contains the complete Unity project, including gameplay systems, assets integration, and the custom architecture developed during the event.

This project was created during the **Sevilla Game Jam 2026**, a game development event where multidisciplinary teams collaborate to design and build a complete game in just a few days.

---

## Features

* Endless runner gameplay
* Dual-dimension mechanic (Physical / Mirror)
* Procedural level generation
* Object pooling system
* Scriptable Object architecture
* Scriptable Event communication system
* Modular game services using the Service Locator pattern
* State Machine based game flow
* Score, health and obstacles systems
* Unity Input System support

---

## Built With

* Unity 6
* C#
* Universal Render Pipeline (URP)
